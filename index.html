Pneumonia Detection

from X-Ray Images using CNN
This project focused on detecting pneumonia in X-ray images using Convolutional Neural Networks (CNNs). The model was trained and evaluated with precision, recall, and F1-score metrics, emphasizing minimizing false negatives. This work aims to improve early detection and management of pneumonia.
